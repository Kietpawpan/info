ระบบติดตามคำขอข้อมูลข่าวสาร (e-Tracking)

MNRE-SLC-008

คู่มือผู้ใช้งานระบบ e-Tracking

ระบบติดตามคำขอข้อมูลข่าวสารของกระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม (e-Tracking) ใช้ตรวจสอบสถานะของคำขอข้อมูลข่าวสารที่ยื่นผ่านระบบ e-Request เพื่อติดตามความก้าวหน้าว่าคำขอนั้น มีการดำเนินการถึงขั้นตอนใดแล้ว อาทิ

  1. ได้รับคำขอแล้ว
  2. ตรวจสอบและรวบรวมข้อมูลข่าวสารตามคำขอ
  3. อยู่ระหว่างการพิจารณา
  4. เปิดเผย (หรือปฏิเสธการเปิดเผย) พร้อมนัดหมายวันรับข้อมูลข่าวสารและแจ้งค่าธรรมเนียม (ถ้ามี)

หลังจากยื่นคำขอผ่านระบบ e-Request แล้ว ท่านสามารถติดตามสถานะคำขอผ่านระบบ e-Tracking ในวันถัดไป


ขั้นตอนการติดตามคำขอ เป็นไปตาม flowchart เริ่มจากเข้าสู่ระบบทาง http://lib.mnre.go.th/info/html/track.php แล้วกรอกรหัสผ่าน เป็นหมายเลขประจำตัวประชาชน 13 หลักของท่าน แล้วกดปุ่ม "ยืนยัน" จากนั้น ระบบจะคุ้มครองข้อมูลส่วนบุคคล โดยการเข้ารหัสข้อมูลดังกล่าว เป็น Public Key ปรากฏในกล่องข้อความถัดจากช่องที่กรอกหมายเลขประจำตัวประชาชน จากนั้นกดปุ่ม "ค้นหา*" ระบบจะสืบค้นและแสดงข้อมูลสถานะของคำขอนั้นทันที ทั้งนี้ กรณีไม่พบรหัสคำขอที่กรอก ระบบจะแจ้งว่า No data! แสดงว่ารหัสไม่ถูกต้องหรือเจ้าหน้าที่ยังมิได้ปรับปรุงข้อมูลในระบบ e-Tracking เนื่องจากอยู่ระว่างรับคำขอหรือรหัสนั้นมิได้อยู่ในสารบบแต่อย่างใด อนึ่ง กรณีมีปัญหาการใช้งาน โทร. 0 2265 6223-5 ตั้งแต่เวลา 8.00 น. - 17.00 น.
