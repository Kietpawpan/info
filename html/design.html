ระบบ e-Request

MNRE-SLC-007

ออกแบบระบบ
e-Request

ระบบรับคำขอข้อมูลข่าวสารของกระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม (e-Request) เขียนด้วยภาษาฝั่งผู้ใช้งาน (client-side script) คือ HTML CSS และ JavaScript เพื่อสร้างบริการอิเล็กทรอนิกส์ ประเภทเว็บแอพลิเคชัน สามารถเข้าถึงและใช้งานบนอุปกรณ์มือถือและเครื่องคอมพิวเตอร์ทั่วไปที่มีโปรแกรมเว็บบราวเซอร์ ผ่าน URL http://lib.mnre.go.th/info และระยะต่อไป อาจย้ายไปอยู่ในเครื่องแม่ข่ายบนคลาวด์กลางของรัฐบาล ภายใต้โครงการ Government Data and Cloud Center (GDCC) ของสำนักงานคณะกรรมการดิจิทัลเพื่อเศรษฐกิจและสังคมแห่งชาติ ซึ่งสำนักงานปลัดกระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม โดยศูนย์บริการร่วมกระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม ได้ยื่นคำขอไว้ในปี พ.ศ. 2567

การออกแบบระบบ e-Request คำนึงถึงการปฏิบัติตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 โดยแจ้งประกาศความเป็นส่วนตัว (privacy notice) ผ่านลิงค์ที่อยู่ด้านบนของหน้าต่างโปรแกรม และมี e-Form สำหรับกรอกคำขอข้อมูลข่าวสาร เมื่อครบถ้วนแล้ว มีปุ่ม "สร้างคำขอ " เพื่อจัดทำร่างคำขอให้ปรากฏในกล่องข้อความแบบอัตโนมัติ (โดยเรียกใช้ฟังก์ชั่นในไฟล์ iRequest.js) เพื่อให้ผู้ใช้งานตรวจสอบความถูกต้อง โดยสามารถพิมพ์แก้ไขเพิ่มเติมข้อความได้ตามความเหมาะสม เมื่อเรียบร้อยแล้ว มีปุ่ม "คัดลอก" และปุ่ม "iOS" หรือ "Gmail" นำผู้ใช้ไปยังหน้าอีเมล เพื่อวางข้อความที่คัดลอกไว้ แล้วส่งอีเมลตามปกติ คำขอจะถูกส่งทางอีเมล ไปยังหน่วยงานตามที่ผู้ใช้งานเลือกไว้

นอกจากนี้ มีปุ่ม "Tracking" ใช้สำหรับเรียกดูข้อมูลสถานะคำขอจากฐานข้อมูลของศูนย์บริการร่วมกระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม
